# simple-blog-django
Simple Blog Django

Clone project.
```bash
git clone https://github.com/silvajunior/simple-blog-django.git
cd simple-blog-django
```
Create and activate venv.
```python
python -m venv venv
source venv/bin/activate
```
Go to blog.
```bash
cd blog
```
Install.
```python
pip install -r 
```
Run.
```python
python manage.py runserver
```
